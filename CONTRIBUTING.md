# Contributing to za_chess_bot

Thank you for your interest in contributing to **za_chess_bot**! We welcome contributions of all kinds—bug reports, enhancements, documentation, and more.

## How Can I Contribute?

### Reporting Bugs
- Use [GitHub Issues](https://github.com/SolamateanTehCoder/za_chess_bot/issues) to report bugs.
- Please include as much detail as possible (OS, environment, steps to reproduce, expected/actual behavior).

### Requesting Features
- Submit feature requests as issues.
- Clearly describe the desired enhancement and its rationale.

### Submitting Pull Requests
- Fork the repository and create your branch from `main`.
- If your change is significant, open an issue first to discuss.
- Write clear, concise commit messages.
- Add or update relevant tests.
- Ensure code style is consistent with the project.
- Document your changes in the pull request description.
- Link the relevant issue (if applicable).

### Coding Standards
- Follow the current code style and structure.
- Comment code where helpful.
- Write tests for new features or fixes.

### Code Review
- All code submissions are reviewed before merging.
- Be responsive to feedback and update your PR as needed.
